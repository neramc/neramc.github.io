🔒 개인정보 처리방침

1. 수집하는 데이터

스케줄러 봇은 서비스 제공을 위해 다음 정보를 수집합니다:

  • 서버 ID (Guild ID): 서버별 일정 및 알람 관리
  • 사용자 ID (User ID): 일정 생성자 식별 및 통계
  • 일정 정보: 제목, 날짜, 시간, 설명, 이미지 URL
  • 알람 정보: 시간, 메시지, 반복 설정
  • 채널 ID: 알람 및 일정 알림 전송 위치

2. 데이터 사용 목적

  • 일정 및 알람 관리 서비스 제공
  • 사용자 통계 기록 (생성한 일정/알람 수)
  • 예정된 일정 및 알람 알림 전송

3. 데이터 저장 및 보안

  • 저장 방식: SQLite 데이터베이스에 로컬 저장
  • 보안: 암호화된 서버에서 안전하게 관리
  • 제3자 공유: 절대 제3자와 데이터를 공유하지 않습니다
  • 외부 전송: 데이터는 외부로 전송되지 않습니다

4. 데이터 보유 및 삭제

  • 사용자가 직접 일정/알람 삭제 가능 (즉시 삭제됨)
  • 봇이 서버에서 제거되면 해당 서버의 모든 데이터 삭제
  • 데이터 삭제 요청 시 7일 이내 완전 삭제

5. 사용자 권리

사용자는 다음 권리를 가집니다:

  • 자신의 데이터 열람 권리
  • 데이터 삭제 요청 권리
  • 데이터 처리 중지 요청 권리

6. 쿠키 및 추적

스케줄러 봇은 쿠키나 추적 기술을 사용하지 않습니다.

7. 만 13세 미만 사용자

Discord의 정책에 따라 만 13세 미만 사용자의 데이터는 수집하지 않습니다.

8. 정책 변경

개인정보 처리방침이 변경될 경우, 이 페이지를 통해 공지됩니다.

9. 문의

개인정보 처리방침 관련 문의사항이 있으시면 언제든지 연락해주세요.
