엔더 드래곤 잡는 방법 (Minecraft: Java Edition)

준비 → 엔더 포탈 찾기 → 엔더로 이동 → 크리스탈 파괴 → 드래곤 처치까지, 한 번에 정리!

목표 엔더 드래곤을 안전하고 빠르게 처치

핵심은 딱 2개야: ① 엔드 크리스탈부터 제거, ② 드래곤이 내려왔을 때(퍼치) 확실히 때리기. 자, “보스전은 준비가 80%”라는 말… 진짜야. 🙂

1) 출발 전 준비물 체크리스트

  • 방어구: 다이아 이상(가능하면 네더라이트) + 보호(Protection) 인챈트
  • 무기: 검(날카로움) + 활(힘/무한 or 수리)
  • 화살: 충분히 (무한 인챈트면 1개도 OK)
  • 블록: 2~3스택(코블스톤/원목 등) — 다리 놓기/기둥 세우기
  • 엔더 진주: 떨어질 때 생존용 “비상탈출 버튼”
  • 물 양동이: 낙하 데미지 방지(물착지), 엔더맨 견제에도 도움
  • 느린 낙하(Slow Falling) 포션: 있으면 난이도 확 내려감
  • 회복: 스테이크/황금사과(가능하면) + 즉시 회복 포션(선택)
  • 침대: 리스폰용(엔더에서는 침대 폭발! “리스폰” 목적으로는 엔더 밖에서만)
  • 호박(선택): 머리에 쓰면 엔더맨이 덜 화남(시야가 답답한 게 단점)

주의 엔더에서는 침대/리스폰 앵커가 폭발해!

2) 엔더 포탈 열기 (요약 루트)

  1. 블레이즈 막대 얻기(네더 요새에서 블레이즈 처치)
  2. 엔더 진주 얻기(엔더맨 처치/거래 등)
  3. 조합: 블레이즈 가루 + 엔더 진주 → 엔더의 눈
  4. 지상에서 엔더의 눈 사용 → 요새(Stronghold) 방향 추적
  5. 요새 내부에서 포탈 룸 찾기 → 포탈 프레임에 눈을 채워 엔더 포탈 활성화

기억 꿀팁: “눈(엔더의 눈) = 길찾기 + 문열기” (요새도 찾고, 포탈도 열고!)

3) 엔더 들어가자마자 해야 할 일 (생존 1분 루틴)

  1. 주변 확인: 바닥이 작은 섬이면 먼저 블록으로 안전지대 만들기
  2. 엔더맨 눈 마주치지 않기: 시선은 바닥/수평선 아래로
  3. 물 양동이/블록를 바로 단축키에 둬서 비상 상황 대비
  4. 가능하면 느린 낙하 포션을 미리 마셔두기(낙사 위험 크게 감소)

팁 엔더 드래곤전에서 “죽음 1순위”는 사실 드래곤이 아니라 낙사야.

4) 1단계: 엔드 크리스탈부터 부수기 (최우선!)

엔드 크리스탈은 드래곤을 회복시켜. 그래서 전투 공식은 이거야: “회복 끊고 → 본체 딜”

  • 방법 A: 활로 파괴 (가장 안전)
    • 기둥 위 크리스탈을 멀리서 활로 쏘기
    • 철창(케이지) 있는 크리스탈은 각도 잡아 틈 사이로 맞추거나, 기둥에 올라가서 처리
  • 방법 B: 기둥에 올라가 근접 파괴 (위험)
    • 블록으로 기둥을 쌓아 올라가기
    • 위에서 크리스탈 파괴 후 물착지/엔더 진주로 안전하게 내려오기

주의 크리스탈은 폭발해! 너무 가까이서 부수면 본인도 같이 터질 수 있어.

5) 2단계: 드래곤 딜 넣는 타이밍 (퍼치 때가 찬스)

드래곤이 중앙의 베드락 기둥(포탈 위) 근처로 내려와 머리를 고정하는 순간을 퍼치(Perch)라고 해. 이때가 “무료 딜 타임”!

  • 퍼치 중 공격:
    • 가능하면 머리를 때리기(더 효율적)
    • 검으로 연속 타격 + 치명타 점프는 상황 봐서(낙사/넉백 주의)
  • 공중 비행 중:
    • 활로 견제(특히 드래곤이 멀리 있을 때)
    • 드래곤 브레스(보라색 구름)는 절대 밟지 말기

팁 “내려오면 칼, 날면 활” — 이거 하나만 외워도 전투가 쉬워져.

6) 엔더 드래곤 브레스/넉백 대처법

  • 브레스(보라색 구름): 닿으면 계속 아픔 → 바로 벗어나기
  • 넉백: 기둥 근처에서 튕겨나가면 낙사 위험 → 중앙 가장자리 조심
  • 낙사 직전: 엔더 진주를 섬 쪽으로 던지기(비상 탈출)
  • 물 양동이: 떨어지기 직전에 물 설치해서 착지(물착지)

스스로에게 질문 하나! “내가 지금 죽을 가능성이 큰 건, 적의 공격일까? 아니면 떨어질까?”
답이 “떨어질 것 같다”면, 딜 욕심 버리고 포지션부터 잡는 게 이겨.

7) 처치 후 보상 & 다음 단계

  • 경험치(엄청 많이)가 쏟아짐
  • 중앙에 엔드 포탈 생성(오버월드로 귀환)
  • 근처에 엔더 드래곤 알이 생성됨(장식/기념품)
  • 외곽에 엔드 게이트웨이가 열려 엔드 시티(엘리트라/셔커 상자) 탐험 가능

기억 드래곤 잡은 뒤가 진짜 엔더 “2막”이야. (엘리트라가 기다림!)

8) 초보자용 “안전한 진행 순서” 한 줄 요약

안전지대 만들기 → 크리스탈 전부 파괴(활 우선) → 퍼치 때 검으로 집중딜 → 낙사만 조심!
